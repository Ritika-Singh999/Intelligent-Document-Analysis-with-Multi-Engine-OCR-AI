# Contributing guidelines

Thanks for considering contributing! Please follow these steps:

1. Fork the repository.
2. Create a branch for your feature or bugfix.
   - `git checkout -b feat/your-feature`
3. Make your changes, add tests if applicable.
4. Ensure `flake8` and `pytest` pass.
5. Submit a PR to `main`.

Be sure to reference issues where appropriate and provide a clear description of your changes.
